# 制限と制約

＃＃ 環境

CARET は以下の環境のみをサポートします。

- 単一ホストのサポート
- FastDDS または CycloneDDS をサポート
- Linux、特に Ubuntu のみをサポート
- Humble、Iron、Jazzy をサポート
  - Iron のサポートは実験的な機能です
  - Jazzy のサポートは実験的な機能です
- CARET ライブラリを使用してアプリケーションを再構築する必要があります

## ROS 2 関数

CARETは以下の機能をサポートしていません。

- /rosout および /parameter_even トピック
- サービス
- アクション

＃＃ 実装

CARET は、次のような実装をサポートしていません。

- フルネームが同じ複数のノード
- ROS レイヤーのラッパー
- リエントラントコールバックグループ
- ノード上で動作する処理以外
- rclcppライブラリで構築されたノード以外
